Use these templates to create NEW ARTHIR modules or convert older Kansa modules.

The following templates are provided;
 - Get-Binary-Template.ps1        Used to create a module that uses a binary utility or tool
 - Get-Script-Template.ps1        Used to create a module that uses a script like the older Kansa modules
 - Get-Task-Template-Daily.ps1    Used to create a module that creates a Daily Scheduled Task on the system
 - Get-Task-Template-Hourly.ps1   Used to create a module that creates an Hourly Scheduled Task on the system
 - Get-Zip-Template.ps1           Used to create a module that unzips a file on the system
